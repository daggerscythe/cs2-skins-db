This is my final project for a Database course. 
I ran a Python script that used my Steam API key to fetch all of the items in my 
Counter-Strike 2 inventory and stored them in a .json file. I then parsed the data 
and converted it into an SQL insert script and populated the MariaDB tables.
Using PHP, I made a web interface to display and edit the database of my skins. 
It has basic CRUD functions, specific queries for the database that were 
required in the final project criteria, and a live market price integration.
